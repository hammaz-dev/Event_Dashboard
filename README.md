# 🎯 Smart Event Dashboard

## 📌 Project Overview

Smart Event Dashboard is a simple web-based application built using **HTML, CSS, and JavaScript** that allows users to create and manage events easily.

Users can add events with a title, date, category, and description. The dashboard dynamically displays the events as cards and allows users to delete events or clear all events.

This project also demonstrates **DOM Manipulation, Event Handling, and Dynamic UI updates using JavaScript**.

---

## 🚀 Features

* Add new events with:

  * Event Title
  * Event Date
  * Category
  * Description
* Display events dynamically as **event cards**
* Delete individual events
* Clear all events with one click
* Add sample events automatically
* Responsive layout using **Flexbox and Grid**
* Demonstrates **DOM Manipulation with JavaScript**

---

## 🛠 Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive layout
* **JavaScript (ES6)** – Logic and interactivity
* **DOM Manipulation** – Dynamic element creation and deletion

## ⚙ How It Works

1. User fills the **Add Event Form**.
2. When the **Add Event** button is clicked:

   * JavaScript captures the form data.
   * A new **event card** is created dynamically.
3. The event card displays:

   * Title
   * Date
   * Category
   * Description
4. Users can:

   * Delete a specific event
   * Clear all events
   * Add predefined sample events
